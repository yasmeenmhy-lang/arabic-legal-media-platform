# منصة إدارة وتمكين الحضور الإعلامي والإعلاني للمحامين

منصة عربية RTL مبنية بـ Next.js لإدارة المحتوى الإعلامي والإعلاني للمحامين، مع دعم التخطيط، المراجعة المهنية، النشر، الحملات، التحليلات، التوصيات، ومؤشرات القطاع المجمعة.

## النشر على Vercel

ابدأ من:

[DEPLOYMENT_CHECKLIST.md](./DEPLOYMENT_CHECKLIST.md)

## إعداد العرض الأول

النسخة الأولى على Vercel لا تحتاج إلى قاعدة بيانات.

المنصة تستخدم بيانات تجريبية وخدمات mock، لذلك لا تحتاج إلى:

- PostgreSQL
- SQLite على Vercel
- Docker
- Prisma migrations
- كلمات مرور قاعدة بيانات

## متغيرات Vercel المطلوبة

```env
AI_PROVIDER=mock
NEXT_PUBLIC_DEMO_MODE=true
```

لا تضف `DATABASE_URL` للعرض الأول على Vercel.

## المستخدمون التجريبيون

كلمة المرور لكل المستخدمين:

```text
Demo@12345
```

- lawyer@example.com
- supervisor@example.com
- admin@example.com

## الضوابط المهنية

المنصة لا تنشئ ترتيبا للمحامين، ولا درجات سمعة، ولا تقييما عاما فرديا. مؤشرات المشرفين مجمعة فقط، ولا توجد وظائف لإدارة القضايا أو خدمات الترخيص.

## قاعدة بيانات حقيقية لاحقا

بعد قبول العرض الأول، استخدم PostgreSQL للإنتاج الحقيقي من خلال Vercel Postgres أو Neon أو Supabase.
